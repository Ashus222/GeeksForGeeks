2. Leap Year 
Easy Accuracy: 48.86% Submissions: 3977 Points: 2
Given an integer y. Check whether it can represent a leap year or not.

Example 1:

Input:
y = 2020
Output:
True
Explanation:
2020 is leap year as it multilpe of 4 but
not of 100

public static void utility(int y){
    String isLeap = "False";
    
    if( y % 4 == 0 &&  y % 100 != 0){
        isLeap = "True";
    } else if ( y % 400 == 0){
        isLeap = "True";
    } else {
        isLeap = "False";
    }
    
    
    //The line below prints the output
    System.out.println(isLeap);
}
